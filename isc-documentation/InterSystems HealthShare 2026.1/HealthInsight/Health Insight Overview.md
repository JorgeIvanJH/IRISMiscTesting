> **Note:**
> 
> The documentation on this page applies when Health Insight is using the Feeder Gateway architecure. This is known as an AADBQ-based feed.
> 
> The System Index-based architecture for Health Insight is officially deprecated beginning with version 2026.1. The Direct Feed Service architecture for Health Insight is experimental in version 2026.1.
