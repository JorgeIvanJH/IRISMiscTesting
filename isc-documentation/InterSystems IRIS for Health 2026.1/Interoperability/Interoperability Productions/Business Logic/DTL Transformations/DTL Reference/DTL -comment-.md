# DTL <comment>

Adds comments to a DTL transformation.

## Syntax

```
<comment>
   <annotation>
   ...
   </annotation>
</comment>
```

## Attributes

None.

## Elements

<table><tr><th>Element</th><th>Purpose</th></tr><tr><td><a href="https://docs.intersystems.com/irisforhealthlatest/csp/docbook/DocBook.UI.Page.cls?KEY=EDTL_annotation">&lt;annotation&gt;</a></td><td>A text string that contains the comment.</td></tr></table>

## Description

The contents of the <annotation> element of <comment> appear in the Management Portal to describe the DTL actions.
