# Resetting a FHIR Server

You can reset a FHIR server. This deletes all data from the server but leaves the server itself intact.

> **Note:**
> 
> You can also reset a FHIR server using the FHIR Server Management REST API.

Follow these steps to reset a FHIR server:

1.  Open the `FHIR Server Management` page.
    
    Either of the following paths in the Management Portal will take you to the page:
    
    *   Navigate to `Home` > `Health` (or `Home` > `HealthShare`) and click `FHIR` in the banner.
        
    *   Navigate to `Home` > `Health` > `FHIRServerNamespace` > `FHIR Server Management`.
        
2.  In the tile for the server associated with the desired endpoint, open the menu and choose `Reset`.
    
3.  In the `Reset FHIR Data` dialog that appears, click `Execute`.
